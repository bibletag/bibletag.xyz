What is the BibleTag API?

A free JSON API facilitating collaborative tagging of the Bible.

Verses From Tags

Instead of searching for verses by reference, pass a tag (e.g., love, worry, etc.) to the BibleTag API and it will return a passage of scripture relevant to that tag.

Collaborative Tagging

The BibleTag API’s tags are crowdsourced from those that use the API. Submit your tags via the POST to contribute to and improve the repository of tags.

Open and Free

Our hosted API is free for your use. See Usage below to get started! Also, the source code is open and we maintain a public Docker image, so you can deploy your own BibleTag API at your organization.
